## Structured Data Classification Project

Notebook-based ML project: data cleaning + feature engineering + EDA + model comparison (5-fold CV).

### How to run
pip install -r requirements.txt
Open `notebook.ipynb` and run all cells.

### Contents
- Data cleaning: unit parsing, one-hot/multi-hot encoding, imputation
- EDA: boxplots, scatter, correlation heatmap
- Models: Logistic Regression, KNN, Decision Tree + CV
